# 数据库迁移与使用命令

本文档只保留在另一台电脑上恢复数据库并启动项目所需的最短命令。

已导出的数据库文件名：

- `ai_opportunity_score_20260412_215313.sql`

建议先把下面这些内容一起拷到另一台电脑：

- 项目源码目录
- `ai_opportunity_score_20260412_215313.sql`
- `ai-opportunity-agent/.env`

## 方案一：目标机器继续用 Docker MySQL

### 1. 启动 MySQL 容器

```bash
docker run -d --name ai-opportunity-mysql \
  -e MYSQL_ROOT_PASSWORD=root \
  -e MYSQL_DATABASE=ai_opportunity_score \
  -p 3306:3306 \
  mysql:8.4
```

### 2. 等待数据库启动完成

```bash
docker logs -f ai-opportunity-mysql
```

看到 `ready for connections` 后停止查看日志。

### 3. 导入数据库

如果你在 PowerShell 中执行：

```powershell
Get-Content ".\ai_opportunity_score_20260412_215313.sql" | docker exec -i ai-opportunity-mysql mysql -uroot -proot
```

如果你在 Git Bash 中执行：

```bash
docker exec -i ai-opportunity-mysql mysql -uroot -proot < "./ai_opportunity_score_20260412_215313.sql"
```

### 4. 验证导入成功

```bash
docker exec -it ai-opportunity-mysql mysql -uroot -proot -e "USE ai_opportunity_score; SHOW TABLES;"
```

```bash
docker exec -it ai-opportunity-mysql mysql -uroot -proot -e "USE ai_opportunity_score; SELECT COUNT(*) AS lead_count FROM lead_info;"
```

## 方案二：目标机器本机已安装 MySQL

### 1. 创建数据库

```bash
mysql -h 127.0.0.1 -P 3306 -u root -p -e "CREATE DATABASE ai_opportunity_score CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
```

### 2. 导入数据库

```bash
mysql -h 127.0.0.1 -P 3306 -u root -p ai_opportunity_score < ai_opportunity_score_20260412_215313.sql
```

### 3. 验证导入成功

```bash
mysql -h 127.0.0.1 -P 3306 -u root -p -e "USE ai_opportunity_score; SHOW TABLES;"
```

```bash
mysql -h 127.0.0.1 -P 3306 -u root -p -e "USE ai_opportunity_score; SELECT COUNT(*) AS lead_count FROM lead_info;"
```

## 导入后如何启动项目

如果目标机器数据库地址仍然是 `127.0.0.1:3306`，用户名密码仍然是 `root/root`，后端通常不用改配置。

### 1. 启动后端

在 `ai-opportunity-backend` 目录执行：

```bash
.\mvnw.cmd spring-boot:run
```

### 2. 启动前端

在 `ai-opportunity-frontend` 目录执行：

```bash
npm install
npm run dev
```

### 3. 访问前端

浏览器打开：

```text
http://localhost:5173
```

## 如果数据库账号或端口不同

启动后端前先设置环境变量：

```powershell
$env:APP_DB_URL="jdbc:mysql://127.0.0.1:3306/ai_opportunity_score?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai"
$env:APP_DB_USERNAME="root"
$env:APP_DB_PASSWORD="root"
.\mvnw.cmd spring-boot:run
```
